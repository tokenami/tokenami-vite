# Vite + React + Tokenami

An example project using [Tokenami](https://tokenami.com) with [Vite](https://vitejs.dev/) and [React](https://reactjs.org/).

If you're using a VS Code-based editor, add the following settings to `.vscode/settings.json` to enable [suggestions for partial strings](https://github.com/tokenami/tokenami#enable-string-completions):

```json
{
  "editor.quickSuggestions": {
    "strings": true
  }
}
```
